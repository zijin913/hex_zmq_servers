# Multi Launch Example

## Description

This example demonstrates how to combine multiple launch configurations into a single launch script. It shows the advanced usage of dynamically loading node configurations from different launch files and merging them together.

## Structure

```bash
multi_launch/
├── launch.py  # multi-launch script
└── README.md  # this file
```

## Dependencies

### Hardware

None.

### Environment

None.

## Usage

1. Activate the virtual environment

    ```bash
    cd path/to/hex_zmq_servers
    source .venv/bin/activate
    ```

2. Run the launch script

    Run the launch script:

    ```bash
    cd examples/adv/multi_launch
    python launch.py
    ```

    The output should be like this:

    ```bash
    ...
    ```
